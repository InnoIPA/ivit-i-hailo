class Source (input: str, resolution: tuple = None, fps: int = None)

Ivit Source Object

  1. Keep update the newest frame
  2. Keep loop in background
  3. Auto reload source

Args - input - the input data, supported image path, image folder, video path, rtsp uri and usb camera - resolution - A tuple value with width and height, you will get target size of the frame after setting up. The default value is None. - fps - This option is only for camera. - start - Start the source piepline after instance.
